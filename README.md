# StudentOS AI — Working Prototype

แอปจริงสำหรับ **Samsung Solve for Tomorrow 2026**
AI Decision Assistant สำหรับนักเรียนมัธยม — "รู้ว่าต้องทำอะไรก่อน เสมอ"

> เวอร์ชันนี้**ทำงานได้จริง** ไม่ใช่ mockup: เพิ่ม/แก้/ลบ/ติ๊กเสร็จงานได้จริง,
> ข้อมูลเก็บในเครื่อง (localStorage), AI อ่านข้อความ/รูปใบงานได้จริง

## วิธีเปิด

- ดับเบิลคลิก `index.html` หรือใน VS Code ใช้ **Live Server** (คลิกขวา → Open with Live Server)
- **Scan รูปด้วย OCR ต้องเปิดผ่าน Live Server / http** (เปิดจากไฟล์ตรง ๆ ใช้ได้เฉพาะแปะข้อความ)

## ฟีเจอร์ที่ทำงานจริงแล้ว

| ฟีเจอร์ | ทำงานยังไง |
|---|---|
| 🤖 AI อ่านข้อความ | แปะข้อความจาก LINE → parser ภาษาไทยแกะ วิชา / ครู / deadline ("พรุ่งนี้", "วันศุกร์", "25 ก.ค.", "16:00") / คะแนน (%) / เวลาที่ใช้ ("ข้อ 1–10" → ประเมินนาที) |
| 📷 Scan รูปใบงาน | OCR ในเบราว์เซอร์ (Tesseract.js, ไทย+อังกฤษ) → ส่งเข้า parser เดียวกัน — ครั้งแรกต้องดาวน์โหลดโมเดล |
| ✔ หน้ายืนยัน | ทุกงานที่ AI อ่าน ต้องผ่านฟอร์มตรวจสอบก่อนเสมอ (แก้ที่ AI อ่านผิดได้) |
| 🔥 จัดลำดับ + เหตุผล | คะแนน priority = ความด่วน (ชม.ที่เหลือ) + น้ำหนักคะแนน + ขนาดงาน + โบนัสสอบ → Top 3 พร้อมเหตุผลภาษาไทยกำกับทุกงาน |
| 🗣 AI พูดก่อน | ประโยคแนะนำสร้างจากข้อมูลจริง: งานไหนก่อน เพราะอะไร เวลาว่างพอไหม |
| 🗓 Timeline | จัดกลุ่ม เลยกำหนด/Today/Tomorrow/This Week + ตรวจ "วันงานชน" แล้วเตือนล่วงหน้า |
| 🔔 แจ้งเตือน | Notification จริงเมื่องานเหลือ < 24 ชม. (ขณะเปิดแอป) |
| 💾 ข้อมูล | localStorage — ปิดเปิดใหม่ข้อมูลอยู่ครบ · มีปุ่มโหลดข้อมูลตัวอย่างสำหรับ demo ในแท็บ "ฉัน" |

## โครงสร้างไฟล์

```
StudentOS-AI/
├── index.html   # ทุกจอ: Scan, ฟอร์มยืนยัน/แก้ไข, Dashboard, Assignments, Timeline, ฉัน
├── engine.js    # สมองของแอป: parser ภาษาไทย + priority engine + ประโยค AI
├── app.js       # UI + state + localStorage + OCR + แจ้งเตือน
├── style.css    # Design system ขาว+น้ำเงิน One UI (บนมือถือแสดงเต็มจอ)
├── docs/strategy.html  # เอกสาร Product Strategy v0.1
└── README.md
```

## สคริปต์ Demo (30 วินาที)

1. เปิดแอปครั้งแรก → เริ่มที่หน้า **Scan** (จุดขาย)
2. แปะข้อความจริงจาก LINE หรือถ่ายรูปใบงานจริง
3. AI แกะข้อมูล → หน้า**ยืนยัน**โชว์ว่าเจออะไรบ้าง (จุดที่โชว์กรรมการว่าเราคิดเรื่อง AI อ่านผิดมาแล้ว)
4. บันทึก → Dashboard: **AI พูดก่อน** + งานขึ้น Top 3 พร้อมเหตุผล

ประโยคทดสอบที่ AI อ่านได้แน่นอน:
```
การบ้านฟิสิกส์ ครูสมชาย ทำโจทย์บทที่ 4 ข้อ 1-10 ส่งพรุ่งนี้ 16:00 คะแนนเก็บ 20%
```
